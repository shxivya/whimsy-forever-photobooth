# ✦ whimsical forever photobooth

a dreamy little web photobooth made for fun ♡

i built this because i wanted the internet to feel softer, prettier, and a little more whimsical.  
so this is a tiny corner where people can capture silly little memories and turn them into tiny forever things.

---

## ✧ features

- vintage-inspired photobooth experience
- countdown capture sequence
- dreamy flash effect
- downloadable photo strips
- soft whimsical UI
- interactive landing page

---

## ✧ built with

- Next.js
- React
- Tailwind CSS
- html2canvas

---

## ✧ why i made this

i love websites that feel personal and magical instead of cold and overly polished.

this project was basically me asking:

> what if taking webcam pictures online felt like keeping memories in a scrapbook?

so i made this ♡
